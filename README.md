# To-Do List C++

Uma aplicação de lista de tarefas (_To-Do List_) desenvolvida em C++ para rodar diretamente no terminal. O projeto foi estruturado utilizando conceitos de Programação Orientada a Objetos (POO).

![CMake](https://img.shields.io/badge/CMake-green.svg)

---

## 🛠️ Tecnologias e Ferramentas

- **Linguagem:** C++
- **Sistema de Build:** CMake

---

## 🚀 Como Executar o Projeto

Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina Linux (ou ambiente compatível):

- Compilador C++ suportando o padrão C++20 (como o g++ ou clang).
- Utilitário CMake.
- Gerenciador de automação make.

Passo a Passo para Compilação

1. Clone este repositório em sua máquina local:

   ```bash
    git clone [https://github.com/danilocamposdev/to-do-list.git](https://github.com/danilocamposdev/to-do-list.git)
    cd to-do-list
   ```

2. Compile o código fonte gerando o executável:

   ```bash
    cmake --build build
   ```

3. Execute a aplicação:

   ```bash
    ./build/to-do-list
   ```
